Expanding Cards

  • Created panels in html using divs and h3 for captions
  • Used inline styling in html for adding pictures
  • In CSS, styled each panel and created a separate class of styling called .active
  • Added click event listeners to each panel in javascript. On-click the .active is added/removed from classlist

Concepts used:

  • Flexbox
  • Transitions
  • Media Queries to make it mobile-responsive

Future Development Ideas

  • Add videos instead of static images
  • Connect to SQL database to make it dynamic
